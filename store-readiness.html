앱 출시 준비 체크리스트

현재 Android debug APK는 빌드 가능하며, Play Store 등록 전 아래 항목을 준비해야 합니다.

Android release

  • Release signing key 생성 및 안전한 보관
  • Release AAB 빌드
  • Play Console 앱 등록
  • 스토어 스크린샷, 아이콘, 소개문, 카테고리 준비

App Links

  • 최종 도메인이 현재 프로젝트로 연결되어야 합니다.
  • Release 인증서 SHA256 fingerprint를 assetlinks.json에 반영해야 합니다.
  • https://seoul-orthopedic.vercel.app/.well-known/assetlinks.json 응답 확인이 필요합니다.

정책 URL

  • 개인정보 처리방침 URL
  • 비급여 안내 URL
  • 의료기관 정보 URL
  • 문의 전화 및 이메일/담당자 정보
